# Code review

To build the extension, enter the following commands inside the project's folder:

1. `pnpm i`
2. `pnpm zip` - to build a bundle of the extension
3. `pnpm dev` - to launch an instance of Chrome with the extension

> For Firefox, use `pnpm zip:firefox` to build a bundle. Development mode with Firefox is not supported.

**Developed and tested on the following machine:**
1. OS: Windows 11 24H2
2. `node`: `22.14.0`
3. `npm`: `10.9.2`
4. `pnpm`: `10.7.0`
